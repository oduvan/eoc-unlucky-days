Пятница 13-го или Черная пятница считается несчастливым днем. Подсчитайте, какое количество таких дней будет в данном году.

Найдите количество Пятниц 13-го в данном году.

Входные данные: Год как integer.

Выходные данные: Число Черных пятниц в году как integer.

Пример:

{% if interpreter.slug == "js-node" %} 
unluckyDays(2015) == 3
unluckyDays(1986) == 1
{% else %} 
unlucky_days(2015) == 3
unlucky_days(1986) == 1
{% endif %}

Предусловие: 1000 < |year| < 3000
